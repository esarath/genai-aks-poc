✅ Best Practices

📈

Autoscaling — Free Tier Safe

Set HPA minReplicas=1, maxReplicas=3. Cluster Autoscaler max 5 nodes (Standard_B2s). Use KEDA for event-driven scaling on queue depth. Always set CPU request = 70% of limit for HPA headroom.

🔐

Security — Zero Secrets in Code

All secrets in Azure Key Vault. CSI driver auto-injects as env vars. DockerHub token in K8s secret. Workload identity for Azure resources. RBAC: namespace-scoped service accounts only. Trivy scans all images in CI.

💰

Cost Optimization — Stay Free

Use Ollama (Phi-3 Mini) on-cluster → $0 LLM cost. Only fall back to Azure OpenAI if needed. Set requests << limits to allow bin-packing. kubectl proxy for Grafana/ArgoCD (no extra public IPs). PVC sizes: Qdrant 10GB, Ollama 20GB, Grafana 5GB.

🎯

RAG Quality

Chunk at 512 tokens with 50-token overlap. Use cosine distance, threshold 0.7. Test with RAGAS: faithfulness >0.8, relevancy >0.8. Use HyDE for complex queries. Re-rank with cross-encoder for high-stakes use cases.

🔄

GitOps Discipline

All Helm values in GitOps repo (esarath/genai-aks-gitops). Never kubectl apply manually in production. Image tag = git SHA, never "latest" in Helm values. ArgoCD auto-sync with self-heal. PRs required for all manifest changes.

📊

LLM Observability

Instrument every inference: latency, token count, model name. Track retrieval scores in histograms. Alert on p99 latency >30s and error rate >5%. Use OpenTelemetry for distributed tracing through RAG chain. Export to Azure Monitor for long-term retention.

🛡

Prompt Injection Protection

Sanitize all user inputs before embedding. Detect injection patterns (ignore previous instructions). Use structured output for API responses. Never expose raw system prompts. Rate-limit by IP/API key.
